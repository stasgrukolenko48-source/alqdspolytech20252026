# Лабораторная работа 1
## Вариант 22: Список дней рождений

### Условие задания:
Файл содержит следующую информацию (построчно): фамилия, имя, отчество, дата рождения. 
Считать файл в список таким образом, чтобы он оставался отсортирован по дате рождения. 
Вывести список фамилий и дат рождения на экран (отсортированным по датам рождения). 
Ввести с клавиатуры дату рождения и проверить, есть ли в списке человек (один или несколько), 
родившийся в этот день.

### Структура проекта:
- `src/birthdays.h` - заголовочный файл
- `src/birthdays.c` - реализация функций
- `src/main.c` - основная программа
- `tests/test_birthdays.c` - unit-тесты
- `data/birthdays.txt` - пример данных

### Компиляция:
```bash
gcc -o birthday src/main.c src/birthdays.c
